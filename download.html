Additional update sites

The update sites for the Additional GEMOC Components can be found in the GEMOC Component Discovery Catalog

This page lists the update sites used to built the Eclipse GEMOC Studio from an empty Eclipse.

Requirements for Eclipse packages and update site: Java 11 .
Getting started: The studio comes with various example languages that you can import in your workspace: File > New > Examples…. — You can also install additional examples using the component discovery service: Help > Install additional GEMOC components (See the current catalog). — Or you can also browse the language gallery for some more examples.

Note for MacOS user: In order to get rid of the quarantine status on the bundle, you need to run xattr -d com.apple.quarantine /path/to/Eclipse.app
