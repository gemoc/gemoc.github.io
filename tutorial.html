In the GEMOC documentation, the DIY section of several of the official examples provides tutorial material:
  • DIY section of the K3FSM official example: The example focuses on the tooling of a single language (FSM) with the following tools: Tree editor, Xtext Editor, Graphical Sirius Editor and Animator, GEMOC Java Engine (Ie. using a Sequential approach), and Model execution with debugging and animation capabilities.
Several tutorials about have been written in order to highlight some of the GEMOC features. Even if some of them have been written using older version of the GEMOC Studio and might not work with the latest version of the studio, they may present many interesting points. This table helps to identify the features presented by the tutorials, their diffusion date and the GEMOC Studio version used.
